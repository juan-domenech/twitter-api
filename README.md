# Twitter-Bot

## Requisites

Python 2.7.x
MongoDB
Twitter Developer account (https://dev.twitter.com/)
tweepy
pymongo

## Description

Simple Twitter-Bot with functions to get your followers list, store them (MongoDB) and send welcome/farewell messages to them.

### twitter_api.py

Twitter functions Class.

### new_followers.py

Main module.

### creds.py
